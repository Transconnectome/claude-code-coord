## 버그 수정 기록 (학습 사항)

### Bug 1: `set -euo pipefail` + `grep` no-match
- **파일**: `memory_scan.sh:24-29`
- **증상**: 한국어 조사만 입력 시 TC-06 exit 1
- **원인**: `pipefail` 환경에서 grep no-match가 command substitution 내에서도 전파
- **수정**: 파이프라인 끝에 `|| true` 추가

### Bug 2: `((PASS++))` bash 산술 exit code
- **증상**: TC-01 이후 테스트 스크립트 조기 종료
- **원인**: `((0))` = exit 1 in bash (PASS=0일 때 old value 0을 평가)
- **수정**: `PASS=$((PASS+1))` 패턴으로 전환

### Bug 3: `set -e` 테스트 내부 재활성화
- **증상**: `set +e / set -e` 쌍이 `set -e` 제거 후에도 남아 충돌
- **수정**: 모든 `set +e / set -e` 쌍 제거
